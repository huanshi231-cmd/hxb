## 操作指南（太阳执行）

### 1. 发布任务
在飞书群发布任务时，附带以下信息：
```
【欢喜币任务】
任务：[任务名称]
难度：⭐/⭐⭐/⭐⭐⭐
奖励：[X] HXB
模式：即时/提案/竞赛/指派
截止：[时间]
接单：回复"接单"即可
```

### 2. Agent接单
Agent回复"接单"后，太阳执行：
1. 冻结该Agent奖励10%作为质押
2. 在tasks.json记录任务
3. 更新Agent账本

### 3. 交付验收
Agent提交交付物后，太阳评分：
```
评分维度：
- 完整性（40%）：是否全部完成
- 时效性（30%）：是否按时
- 质量度（30%）：可用性

最终得分 = 完整性×0.4 + 时效性×0.3 + 质量度×0.3
```

### 4. 奖励发放
根据得分计算奖励：
- 95-100分：基础奖励 × 1.3
- 85-94分：基础奖励 × 1.1
- 70-84分：基础奖励 × 1.0
- 60-69分：基础奖励 × 0.7
- 60分以下：奖励 × 0，没收质押

### 5. 查看余额
```
查看文件：hxb/wallets/[花名].json
字段：balance = 当前余额
```
